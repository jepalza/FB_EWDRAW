# FB_EWDRAW
Librería para apoyo CAD usando FreeBasic

Empleando la libreria CAD/CAE de aqui:

http://www.eastwindsoft.com/es/


He creado un mini-CAD con llamadas a varias rutinas de EWDRAW 


He necesitado utilizar la libreria AXSUITE3 del foro freebasic:

https://www.freebasic.net/forum/viewtopic.php?t=23197&sid=24b2039bbb28a9765e8f761931c87e45


Y con un fichero "manifest" se puede usar la OCX sin necesidad de registro previo usando "regsvr32".

Este código FreeBasic no está completo, es mas una demostración, pero funciona como CAD de solidos 3D.

Se emplea una versión un poco vieja del EWDRAW por ser la mas compatible, en este caso una version 15.9.4 de 2018:

setup_ult_v15_trial.exe

que ya tenía descargada de hace años. Hoy día aún puede encontrarse en una conocida web de descarga de archivos viejos.
